# Security Policy

## Reporting Security Vulnerabilities

We take security seriously at SecureChat. If you discover a security vulnerability, please follow responsible disclosure practices.

### How to Report

1. **Do not** open public issues for security vulnerabilities
2. Email security reports to: [security@your-email.com]
3. Include detailed information about the vulnerability
4. Provide steps to reproduce the issue if possible

### What to Include

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if you have one)
- Your contact information

### Response Timeline

- **Initial Response**: Within 24 hours
- **Status Update**: Within 72 hours
- **Fix Timeline**: Varies based on severity and complexity

### Supported Versions

We provide security updates for the following versions:

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |
| < 1.0   | :x:                |

### Security Best Practices

When using SecureChat:
- Keep the app updated to the latest version
- Use strong authentication methods
- Be cautious with file sharing
- Report suspicious activity

Thank you for helping keep SecureChat secure!
